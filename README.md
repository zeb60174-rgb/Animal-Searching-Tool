# Animal-Searching-Tool

# 🐾 Animalis — Voice & Text Animal Explorer

Animalis is a modern web-based animal search tool that allows users to explore animals using **text input or voice commands**. It fetches real-time data from Wikipedia and displays rich information including images, scientific names, descriptions, and multilingual translations.

---

## 🚀 Features

* 🔍 Search animals by name
* 🎤 Voice recognition support (speak animal name)
* 🖼️ Dynamic animal images
* 📖 Wikipedia-based descriptions
* 🧬 Scientific names extraction
* 🌍 Multilingual animal names
* ⚡ Fast & responsive UI
* ❌ Error handling for invalid searches

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Glassmorphism UI)
* JavaScript (Vanilla JS)
* Wikipedia REST API
* Web Speech API (Voice Recognition)

---

```
Animalis/

```

---

## ⚙️ How It Works

1. User enters an animal name OR uses voice input
2. The app fetches data from Wikipedia API
3. Displays:

   * Animal name
   * Scientific name
   * Description
   * Image
   * Family & diet (inferred)
   * Translations in multiple languages
